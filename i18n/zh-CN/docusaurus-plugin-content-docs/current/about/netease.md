# 区域法律限制

:::info

自4.1.0.4开始，这一限制已经取消。

:::

自 4.1.0.2 以来，QuickShop-Hikari 将不再允许在位于[网易游戏独家代理区域（例如：中国大陆）](https://mc.163.com/)运行。

此限制不能被取消。

有关此内容的更多信息，请参阅：[环境检查](https://github.com/QuickShop-Community/QuickShop-Hikari/blob/hikari/quickshop-bukkit/src/main/java/com/ghostchu/quickshop/util/envcheck/EnvironmentChecker.java)。

此功能将请求 CloudFlare 网络，参见： [隐私政策 -> IP 检测](privacy.md)
