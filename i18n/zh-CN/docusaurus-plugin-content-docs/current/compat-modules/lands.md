## 配置

```yaml
# 是否阻止 Land 插件领地外的商店创建或相关交易？
whitelist-mode: false
# 插件是否忽略被禁用的世界。
ignore-disabled-worlds: true
# 在商店创建者失去对应领地的权限时，是否将他的商店一并移除？
delete-on-lose-permission: false
```
